Guía de configuración

Cómo conectar con Google Sheets

1. Abre Google Sheets y crea una hoja nueva. En la primera fila pon estas columnas exactas:

Fecha | Operador | Unidad | Familia | Componente | Accion | Observaciones

2. En Sheets ve a Extensiones → Apps Script y pega este código:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);

  sheet.appendRow([
    data.fecha,
    data.operador,
    data.unidad,
    data.familia,
    data.componente,
    data.accion,
    data.observaciones,
    new Date()
  ]);

  // Enviar correo de notificación
  var correo = "TU_CORREO@gmail.com"; // ← Cambia esto
  var asunto = "Nuevo reporte: " + data.unidad + " - " + data.familia;
  var cuerpo = "Se recibió un nuevo reporte:\n\n"
    + "Fecha: " + data.fecha + "\n"
    + "Operador: " + data.operador + "\n"
    + "Unidad: " + data.unidad + "\n"
    + "Familia: " + data.familia + "\n"
    + "Componente: " + data.componente + "\n"
    + "Acción: " + data.accion + "\n"
    + "Observaciones: " + data.observaciones;

  MailApp.sendEmail(correo, asunto, cuerpo);

  return ContentService
    .createTextOutput(JSON.stringify({resultado: "ok"}))
    .setMimeType(ContentService.MimeType.JSON);
}

3. Cambia "TU_CORREO@gmail.com" por tu correo real.

4. Haz clic en Implementar → Nueva implementación → Aplicación web.

5. En "Quién puede acceder" elige Cualquier persona. Copia la URL generada.

6. Pégala en el campo "URL del Google Sheet" arriba en este formulario y guarda.
